# Data Dictionary

## Purpose

This document defines the structure, meaning, and expected values of the synthetic behavioral dataset generated by **PROJECT NAVIDSON**.

The data dictionary serves as the canonical reference for feature engineering, behavioral analytics, machine learning, and downstream analysis.

Each record represents a single behavioral observation associated with one employee during a defined observation period.

---

# Dataset Overview

**Observation Unit**

One behavioral observation associated with a single employee during a defined observation period.

**Observation Period**

One business day (current implementation)

**Primary Key**

Employee_ID + Timestamp

**Target Variables**

- Anomaly_Score (continuous)
- Recommended_Action (categorical)
